物理原理与数学表达

两个同方向、同频率的简谐振动合成后仍为简谐振动，其振幅和初相位由原来的两个振动决定。

y₁(t) = A₁·sin(ω₁t + φ₁)
y₂(t) = A₂·sin(ω₂t + φ₂)
y(t) = y₁(t) + y₂(t)

当两个振动频率相同时，合成振动为：

y(t) = A·sin(ωt + φ)
A = √(A₁² + A₂² + 2A₁A₂cos(Δφ))
tanφ = (A₁sinφ₁ + A₂sinφ₂) / (A₁cosφ₁ + A₂cosφ₂)

当频率不同时，合成振动不再是简谐振动，会产生"拍"现象，拍频为两频率之差。

李萨如图形原理

两个相互垂直的简谐振动合成的轨迹称为李萨如图形，其形状由两个振动的频率比和相位差决定。

x(t) = Aₓ·sin(ωₓt + φₓ)
y(t) = Aᵧ·sin(ωᵧt + φᵧ)

李萨如图形可用于测量未知频率：当两个振动的频率比为简单整数比时，图形稳定且具有对称性。频率比等于图形在x轴和y轴方向的切点数之比。

相位差的变化会使图形发生旋转，不同的相位差会产生不同的图形：

  • Δφ = 0°: 直线（第一、三象限对角线）
  • Δφ = 90°: 椭圆（当振幅相等时为圆）
  • Δφ = 180°: 直线（第二、四象限对角线）
